✅ PASSO 1 — Corrigir SQL (prioridade máxima)

Antes de qualquer arquitetura:

1️⃣ Elimine concatenação de string
2️⃣ Use parâmetros em 100% das queries
3️⃣ Remova manipulação manual de datas
4️⃣ Use AsDateTime

Só isso já sobe seu nível técnico visivelmente.

Se um tech lead ver isso, ele já percebe maturidade.

✅ PASSO 2 — Ajustar responsabilidade da Query

Meta:

Repository deve criar e gerenciar sua própria TFDQuery.

Exemplo mental:

function Listar(...): TFDQuery;
var
  Query: TFDQuery;
begin
  Query := TFDQuery.Create(nil);
  ...
  Result := Query;
end;


Ou melhor ainda: retornar lista de objetos.

Mas primeiro organize posse da query.

✅ PASSO 3 — Remover Variant

Trocar:

tipoAtividade: Variant


por

tipoAtividade: Integer;


E usar -1 como “não filtrado”.

Tipagem forte muda seu código de nível.
